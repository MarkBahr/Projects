Making a Simple Quiz

The form tag is important for quizzes or any web form. An important aspect of forms that this quiz uses is the radio button. Each correct answer was given a value of one and the others were given a value of zero. Also, the ability to only choose one correct answer is done by giving each set of radio buttons the same name. These names are referenced in the Javascript for scoring purposes. If you wish to see the source code for this page, simply right click and choose "view page source."

This quiz uses Javascript to scoring capablilty. After answering all the questions, press the submit button and it will show you your score. If you wish to re-take the quiz, simply click "reset" and start over.
